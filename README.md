# Machine Learning Methods

This repository serves as a comprehensive reference for the mathematical foundations and practical implementations of core machine learning algorithms. 

## Topics Covered

### 1. Mathematical Foundations
* **Multivariate Calculus:** Gradient vectors, Jacobians, and chain rule derivations.
* **Linear Algebra:** Eigenvectors, eigenvalues, and matrix operations.
* **Statistical Learning Theory:** Hypothesis spaces, Empirical Risk Minimization (ERM), and PAC learning bounds.

### 2. Supervised Learning
* **Regression:** Linear regression and least squares optimization.
* **Classification:** k-Nearest Neighbors (k-NN) and logistic regression.
* **Optimization:** Gradient Descent and Stochastic Gradient Descent (SGD).

### 3. Unsupervised Learning
* **Clustering:** K-Means clustering and centroid initialization.
* **Dimensionality Reduction:** Principal Component Analysis (PCA) and covariance analysis.

### 4. Deep Learning & Advanced Methods
* **Neural Network Foundations:** Backpropagation and loss functions.
* **Transformers:** Self-attention mechanisms and architecture overviews.

## Repository Structure

* `docs/`: LaTeX notes and mathematical proofs.
* `notebooks/`: Jupyter notebooks containing Python implementations.
